Analysis of binary vapour-liquid equilibria data from Dortmund Data Bank

 chem-engg-tools.herokuapp.com
This web application reads data from DDBST's collection of VLE data for 30 important compounds. Binary vapour-liquid equilibrium data of 30 Important Components from Dortmund Data Bank can be accessed from the web page. It correlates the isothermal data with Margules model, Redlich Kister (truncated to two terms), van Laar model and Wohl's expansion (truncated to one term). UNIQUAC model is implemented for isobaric data.

Multivariable optimisation for design of STHEs

 sthe-optimization.herokuapp.com
This website minimises the total annual cost of STHEs by carrying out a constrained discrete optimisation exercise to optimise seven key variables in design calculations.
